## 🛠️ Kurulum ve Çalıştırma

Projeleri yerel ortamınızda çalıştırmak için aşağıdaki gereksinimleri karşılamanız gerekir:

### Gereksinimler
- Bilgisayarınızda **.NET SDK** (v8.0 veya üzeri önerilir) yüklü olmalıdır. [.NET İndir](https://dotnet.microsoft.com/download)

### Kurulum ve Çalıştırma Adımları
1.  Projeyi klonlayın:
    ```bash
    git clone https://github.com/Kaaner4mir/CSharpLearningSeries.git
    ```
2.  Çalıştırmak istediğiniz konunun klasörüne gidin:
    ```bash
    cd IntroductionToCSharp
    ```
3.  Projeyi terminal üzerinden başlatın:
    ```bash
    dotnet run
    ```
